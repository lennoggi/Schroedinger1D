# Schroedinger1D
A code evolving the 1D Schroedinger equation with any real, time-independent potential via the Lie-Trotter operator-splitting technique through a discrete Fourier transform. The code was built following [these lecture notes](https://virgilio.mib.infn.it/~oleari/public/laboratorio_fis_computaz/ris_num_eq_Schroedinger.pdf).
